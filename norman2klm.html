Clean Comparison

Model-based evaluation: what each model explains and predicts
Model Category What it Explains What it Predicts
Norman Cognitive / descriptive Understanding, errors, feedback, gulfs Nothing (no performance prediction)
GOMS Predictive / analytic Task structure for expert users Relative efficiency
KLM Predictive / quantitative Low-level physical actions Execution time

Key Takeaways

KLM uses seven primitive operators to predict execution time for expert, error-free interaction.

KLM models execution, not planning or interpretation.
